### Diferencias - Comandos del sistema operativo

| Tarea | Git Bash | PowerShell | CMD |
| --- | --- | --- | --- |
| Listar archivos | `ls` | `Get-ChildItem` o `ls` | `dir` |
| Limpiar pantalla | `clear` | `Clear-Host` o `cls` | `cls` |
| Ver contenido | `cat archivo.txt` | `Get-Content archivo.txt` | `type archivo.txt` |
| Crear archivo | `touch archivo.txt` | `New-Item archivo.txt` | `echo. > archivo.txt` |
| Variables de entorno | `export VAR=valor` | `$env:VAR="valor"` | `set VAR=valor` |
