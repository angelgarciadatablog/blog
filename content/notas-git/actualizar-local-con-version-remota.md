¿Cuándo hacer un pull?

```bash
# 1. Antes de empezar a trabajar (actualizar tu código)
git pull

# 2. Cuando un compañero te dice "ya subí los cambios"
git pull

# 3. Antes de hacer push (evitar conflictos)
git pull
git push
```
