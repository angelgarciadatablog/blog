🔧 Habilitación de APIs necesarias (Cloud Functions 2nd Gen)

Para desplegar el pipeline en Cloud Functions 2nd Gen se habilitaron las siguientes APIs en el proyecto youtube-datasets-360:

  1. Cloud Functions API
  2. Cloud Build API
  3. Artifact Registry API
  4. Cloud Run API
  5. Cloud Scheduler API

Estas APIs permiten:

  • Construcción automática del contenedor (Cloud Build)
  • Ejecución sobre infraestructura administrada (Cloud Run)
  • Almacenamiento del contenedor (Artifact Registry)
  • Programación de ejecución automática (Cloud Scheduler)
