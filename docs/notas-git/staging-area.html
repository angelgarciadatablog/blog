Staging Area

Conceptos clave

  1. Working Directory (tu carpeta actual)
    • Aquí están todos los archivos que estás editando.
    • Git ve cualquier modificación, adición o eliminación, pero todavía no la prepara para commit.
  2. Staging Area (Index)git add
    • Es como un “carrito” donde pones los cambios que sí quieres incluir en el commit.
    • Solo los archivos que estén aquí serán parte del commit.
  3. Repository (commit histórico)
    • Solo se actualiza cuando haces git commit
